Merging
time limit per test
1 second
memory limit per test
256 megabytes
input
standard input
output
standard output

You are given two non-empty strings $$$s_1$$$ and $$$s_2$$$, consisting of lowercase English letters.

You want to merge these two strings into a single string in such a way that:

  1. You can choose the order: either $$$s_1$$$ followed by $$$s_2$$$, or $$$s_2$$$ followed by $$$s_1$$$.
  2. If a non-empty suffix of the first string matches a prefix of the second string, the common part should appear only once in the merged string.

Your task is to find the shortest possible merged string following the above rules.

Input

The first line contains the string $$$s_1$$$ ($$$1 \le |s_1| \le 10^3$$$).

The second line contains the string $$$s_2$$$ ($$$1 \le |s_2| \le 10^3$$$).

Both strings consist only of lowercase English letters.

Output

Output a single string — the shortest merged string.If there are multiple shortest strings, output the one that starts with $$$s1$$$.

Examples

Input
abc
xyz
Output
abcxyz
Input
a
a
Output
a
Input
abc
bc
Output
abc
Input
abc
abc
Output
abc
