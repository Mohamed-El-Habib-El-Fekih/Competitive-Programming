You are given two non-empty strings $s_1$ and $s_2$, consisting of lowercase English letters.

You want to merge these two strings into a single string in such a way that:

\begin{enumerate}
    \item You can choose the order: either $s_1$ followed by $s_2$, or $s_2$ followed by $s_1$.
    \item If a non-empty suffix of the first string matches a prefix of the second string, the common part should appear \textbf{only once} in the merged string.
\end{enumerate}

Your task is to find the \textbf{shortest possible merged string} following the above rules.
